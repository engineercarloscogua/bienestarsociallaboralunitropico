# Identidad visual

## Fuente del criterio

Manual revisado: `manual-identidad-unitropico-baja-resolucion (3).pdf`, primeras 25 páginas.

## Lineamientos observados

- Uso de logosímbolo/escudo institucional.
- Colores principales: verde institucional, bronce/oliva, blanco y negro.
- La construcción del logotipo se basa en `Noa LT Std Bold`; para web se usa aproximación segura con Google Fonts.

## Aplicación web actual

- Títulos: `Nunito Sans`.
- Texto: `Source Sans 3`.
- Fondo general claro.
- Sidebar blanco con acento verde.
- Botones principales en verde/bronce.
- Tarjetas blancas con borde suave e imagen superior.

## Paleta implementada

- Verde principal: `#006b5b`
- Verde secundario: `#007c72`
- Verde oscuro: `#004f45`
- Bronce: `#b8a261`
- Bronce oscuro: `#95843d`
- Fondo: `#f7f8f4`
- Superficie: `#ffffff`
